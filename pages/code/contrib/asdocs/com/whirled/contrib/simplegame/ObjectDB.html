getObjectRefsInGroup()method 
public function getObjectRefsInGroup(groupName:String):Array

Returns an Array containing the object refs of all the objects in the given group. This Array must not be modified by client code. Note: the returned Array will contain null object refs for objects that were destroyed this frame and haven't yet been cleaned up.

Parameters
groupName:String

Returns
Array
getObjectsInGroup()method 
public function getObjectsInGroup(groupName:String):Array

Returns an Array containing the SimObjects in the given group. The returned Array is instantiated by the function, and so can be safely modified by client code. This function is not as performant as getObjectRefsInGroup().

Parameters
groupName:String

Returns
