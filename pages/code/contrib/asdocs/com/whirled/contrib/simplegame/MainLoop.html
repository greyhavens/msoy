insertMode()method 
public function insertMode(mode:AppMode, index:int):void

Inserts a mode into the stack at the specified index. All modes at and above the specified index will move up in the stack. (Mode changes take effect between game updates.)

Parameters
mode:AppMode — the AppMode to add
 
index:int — the stack position to add the mode at. You can use a negative integer to specify a position relative to the top of the stack (for example, -1 is the top of the stack).

Call this function before the application shuts down to release memory and disconnect event handlers. Most applications will want to install an Event.REMOVED_FROM_STAGE handler on the main sprite, and call shutdown from there.
