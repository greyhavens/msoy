Classpublic class AVRGameControl
InheritanceAVRGameControl AbstractControl flash.events.EventDispatcher

This file should be included by AVR games so that they can communicate with the whirled. AVRGame means: Alternate Virtual Reality Game, and refers to games played within the whirled environment with your avatar.

Note: The AVRG framework is "alpha" and may be changed in incompatible ways. If you are making an AVRG game, please let us know what you're doing in the AVRG discussion forum: http://first.whirled.com/#whirleds-d_135_r

setHitPointTester()method 
public function setHitPointTester(tester:Function):void

Configures the AVRG with a function to call to determine which pixels are alive for mouse purposes and which are not. By default, all non-transparent pixels will capture the mouse. The prototype for this method is identical to what the Flash API establishes in DisplayObject: testHitPoint(x :Number, y :Number, shapeFlag :Boolean) :Boolean
