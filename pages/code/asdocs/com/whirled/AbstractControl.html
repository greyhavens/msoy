doBatch()method 
public function doBatch(fn:Function, ... args):void

Execute the specified function as a batch of commands that will be sent to the server together. This is no different from executing the commands outside of a batch, but may result in better use of the network and should be used if setting a number of things at once.

Parameters
fn:Function
 
... args

Example 
     _ctrl.doBatch(function () :void {
         _ctrl.net.set("board", new Array(100));
         _ctrl.net.set("scores", new Dictionary());
         _ctrl.net.set("captures", 0);
     });
Note that it guarantees that those events get processed by the server as a unit, but the results will not come back as a unit. So, for instance, when you receive the PropertyChangedEvent for "board", checking the value of "scores" will still return the old value.
