Parameters
amount:int
getAvailableFlow()method 
public function getAvailableFlow():int

Get the amount of flow that's currently available to award to the occupant who has instantiated this game control. In other words, your game is only responsible for granting flow to the occupantId returned by getMyId(). You should grant flow based on the performance in the game, but can also grant flow to non-players.

Returns
int
